# TFG.-Desigualtats-en-salut-percebuda

Aquest repositori conté el codi utilitzat per al Treball Final de Grau titulat:

**“Desigualtats en la salut percebuda a Espanya”**

L’anàlisi es basa en dades de l’Enquesta de Salut d’Espanya (ESdE) 2023 i inclou:
- Preparació i tractament de la base de dades
- Anàlisi exploratòria
- Model de regressió logística binària
- Model logit ordinal 

## Estructura del repositori

- `Apartat 1. Base de dades.Rmd`: lectura, depuració i tractament de les dades  
- `Apartat 2. Anàlisi exploratori.Rmd`: estadística descriptiva i exploratòria  
- `Apartat 3. Model de Regressió binària.Rmd`: model logit binari ponderat  
- `Apartat 4. Model logit ordinal.Rmd`: model logit ordinal ponderat  

## Notes
Les dades originals no s’inclouen al repositori per motius de confidencialitat.
